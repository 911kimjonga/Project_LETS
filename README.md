## 파이널 프로젝트 (Lets)

<br />

### 🎤 소개 
Lets는 Let's Easy Together Study의 약자로 스터디 카페 예약 서비스와 스터디 그룹 커뮤니티를 제공합니다.

<br />

### 📆 프로젝트 기간
2023년 9월 4일 ~ 2021년 09월 24일

<br />

### 👨‍💻👩‍💻 멤버 구성
- 강소영
- 김종원
- 박상훈
- 이한솔
- 이희영

<br />

### 📌 기능
  - #### 1️⃣ 회원 관련 기능
  - #### 2️⃣ 스터디 카페 관련 기능
  - #### 3️⃣ 예약 및 결제 관련 기능
  - #### 4️⃣ 스터디 그룹 관련 기능
  - #### 5️⃣ 게시글 관련 기능
